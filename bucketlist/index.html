Features implemented for this "bucket list" project

  1. <!DOCTYPE html> for html5 doctype
  2. Viewport declaration using <meta name="viewport" content="width=device-width, initial-scale=1"> for proper scaling
  3. Home screen icons and favicon
  4. Form validation
  5. Modernizr
  6. Vendor prefix solution
  7. Polyfills - for geolocation API and Placeholders
  8. Responsive images
  9. Font Icons (Vector based solution)
  10. Javascript Matchmedia
  11. CSS3 Media Queries
  12. CSS Resets (included in twitter bootstrap)
  13. CSS pseudoclasses
  14. CSS Sprites
  15. Mobile Javascript library (twitter bootstrap)
  16. JSON
  17. Client-side storage (localStorage)
  18. Geolocation API
  19. Mobile-specific URLs for device interaction (tel: and mailto:)
  20. Console messages for debugging
  21. Web Analytics
  22. Social Share (twitter)
