Features implemented for this "bucket list" project

  1. <!DOCTYPE html> for html5 doctype
  2. Viewport declaration using <meta name="viewport" content="width=device-width, initial-scale=1"> for proper scaling
  3. Home screen icons and favicon
  4. Form validation
  5. Modernizr
  6. Vendor prefix solution
  7. Polyfills - for geolocation API and Placeholders
  8. CSS3 Media Queries
  9. CSS Resets
  10. CSS pseudoclasses
  11. Mobile Javascript library (jquery mobile)
  12. Client-side storage (localStorage)
  13. Geolocation API
  14. Mobile-specific URLs for device interaction (tel:)
  15. Console messages for debugging
  16. Web Analytics
  17. Social Share (twitter)
